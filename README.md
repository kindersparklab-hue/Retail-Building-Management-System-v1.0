# Retail Building Management System (RBMS) v1.0
Built from scratch on GitHub. Admin-only platform for managing retail buildings.

## Features
- Dashboard, Shops, Rent, etc.
- Material-UI design.

## Setup (Local)
1. Clone: `git clone https://github.com/yourusername/rbms-website-from-scratch.git`
2. Backend: `cd backend && npm install && node server.js`
3. Frontend: `cd frontend && npm install && npm start`
4. Set up PostgreSQL DB and .env.
